# 简介

设置组件之间的间距，避免组件紧贴在一起，拉开统一的空间。

![](https://www-s.ucloud.cn/2022/08/0be32c57fb98050d160f38f335836345_1660730789819.png)

任意尺寸组件可以视场景灵活使用配置项。
